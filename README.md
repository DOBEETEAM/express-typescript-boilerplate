## Installation
Clone this source
```bash
git clone
```
Run `yarn install` to install all dependencies 

Create file `.env` and copy, paste code below. add your secret info or other sensitive info,...
```bash
APP_HOSTNAME=
APP_PORT=
NODE_ENV=
```
Run `yarn dev` to start server at dev environment
